Put [name of the config].json into configs

Example of config:
{
    "server": "irc.chat.twitch.tv",
    "port": 6667,
    "nickname": "example_user",
    "token": "oauth:example_token",
    "channel": "#example_channel",
    "word_to_detect": "!example_word",
    "count_threshold": 2,
    "action_to_take": "!example_action",
    "repeat": 3,
    "cooldown_period": 60,
    "cooldown_after_send": 30,
    "timer_reset_seconds": 30
}

Note:
channel must have # before the name
